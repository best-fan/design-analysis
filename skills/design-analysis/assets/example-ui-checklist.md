## 七、实现建议

### 7.1 组件划分与样式文件

（建议的组件划分、样式文件、开发顺序）

- 组件划分：
  - `Layout/Navbar` - 顶部导航栏组件
  - `Layout/PageHeader` - 页面标题区组件
  - `Dashboard/StatCard` - 统计卡片组件（可复用）
  - `Charts/LineChart` - 折线图组件
  - `Charts/DonutChart` - 环形图组件
  - `List/MemberList` - 成员结算列表组件

- 样式文件：
  - `styles/variables.scss` - 颜色、字体、间距变量
  - `styles/mixins.scss` - 阴影、圆角等 mixin
  - `components/Dashboard/StatCard.module.scss` - 统计卡片样式

- 开发顺序：
  1. 搭建整体布局框架（导航栏、页面标题、内容区）
  2. 实现统计卡片区（数据展示）
  3. 实现图表区（折线图 + 环形图）
  4. 实现成员列表区
  5. 添加交互状态和动画

### 7.2 主流网页设计常识

详见 `implementation-guidelines.md` 获取详细说明。

- 统计卡片使用 Grid 布局，`grid-template-columns: repeat(3, 1fr)`
- 图表区使用 Grid，`grid-template-columns: 2fr 1fr`
- 卡片阴影使用统一 mixin，避免硬编码
- 文字颜色使用变量，便于主题切换

### 7.3 常见错误模式与避免方法

详见 `implementation-common-errors.md` 获取详细说明。

| 错误模式             | 避免方法                              |
| -------------------- | ------------------------------------- |
| 图表容器无明确高度   | 为图表容器设置固定高度或 min-height   |
| 统计数字换行         | 使用 `white-space: nowrap` 或调整字号 |
| Grid 列宽不一致      | 使用 `fr` 单位或 `minmax()`           |

## 九、备注

（坐标系统、响应式、占位策略等）
